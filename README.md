🎮 Tic Tac Toe Game in Java

This project is a console-based implementation of the classic Tic Tac Toe game developed using Java. The game allows two players to play alternately on a 3x3 grid.

🔹 Features:
- Two-player mode (Player X and Player O)
- Interactive console-based UI
- Input validation to prevent invalid moves
- Detection of winning conditions (rows, columns, diagonals)
- Draw detection when the board is full
- User-friendly prompts and error handling

🔹 Technologies Used:
- Java (Core Java)
- Scanner class for user input
- Arrays for board representation

🔹 How It Works:
- The game board is represented using a 2D array.
- Players take turns entering row and column positions.
- The system checks if the move is valid.
- After each move, the game checks for a winner or draw.
- The game continues until a player wins or the board is full.

🔹 Future Enhancements:
- Add GUI using Java Swing or JavaFX
- Single-player mode with AI
- Score tracking system
- Online multiplayer support

 This project is ideal for beginners to understand basic Java concepts like loops, conditionals, arrays, and user input handling.
